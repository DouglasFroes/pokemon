## Getting Started

First,install dependence:

```bash
npm install
# or
yarn
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Third, run the production server:

````bash
npm run build
```&&```
npm run start
# or
yarn build
 ```&```
yarn start
````

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
